# LectureAgent

## 项目描述

这是一个讲座代理项目。

## 快速开始

### 前提条件
- Python 3.8+
- Node.js 14+（如需）

### 安装

```bash
# 克隆项目
git clone <repository-url>

# 进入项目目录
cd lectureagent

# 安装依赖（Python）
pip install -r requirements.txt
```

## 项目结构

```
lectureagent/
├── src/              # 源代码
├── tests/            # 测试文件
├── docs/             # 文档
├── config/           # 配置文件
├── .gitignore        # Git忽略文件
├── README.md         # 项目说明
└── requirements.txt  # Python依赖
```
